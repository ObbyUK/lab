# Obby Lab

![AltText](https://media.giphy.com/media/wAxlCmeX1ri1y/giphy.gif)

### Setup & Play

* Be on node `8.9.4` or greater
* `yarn`
* `yarn make-var-files`
* `yarn start-lang`
* Go to `localhost:5000`

### Deployment

* Set prod `.env` and `settings.js` files
* `yarn`
* `yarn prod-lang` (this will use **pm2** to run the app)
